
# firatilhancomtr

A practice project for learning the MVVM architecture pattern in iOS development using Swift and UIKit.

## Features
- Fetches data from my personal website (firatilhan.com.tr) via a custom PHP/MariaDB backend I built
- Home screen with photo, title, and description
- Photo gallery with city filter and infinite scroll
- About page with HTML content parsing

## Tech Stack
- Swift / UIKit / Storyboard
- MVVM Architecture
- Alamofire (networking)
- SDWebImage (image loading)
